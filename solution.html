Methodology

How the system is built and evaluated

The PSBN Prototype combines wireless infrastructure, transport-layer continuity, and application-layer tools into a unified system. This methodology outlines how each layer is designed and integrated to support resilient connectivity.

System approach

The system is structured across multiple layers, each responsible for a specific aspect of connectivity, continuity, and operational visibility.

Wireless layer

The platform uses a Linux-based wireless environment with Amarisoft to emulate LTE/5G behavior, enabling device attachment, signaling, and data exchange within a controlled testbed.

Transport layer

Multipath TCP (MPTCP) is used to maintain session continuity across multiple network paths, allowing traffic to persist even when one path degrades or disconnects.

Application layer

TAK / WebTAK provides a real-time operational interface, enabling device tracking, messaging, and system monitoring across the network.

Core workflow

The system follows a structured workflow that integrates device connectivity, network behavior, and application-level interaction.

1
Device connection
Devices connect to the wireless environment and establish network sessions through the Amarisoft platform.
2
Session establishment
TCP sessions are created and managed using MPTCP, allowing multiple paths to be utilized simultaneously.
3
Network interaction
Data flows through the network while logs and system behavior are monitored through the Amarisoft interface.
4
Application integration
TAK services provide a shared operational view, allowing users to observe and interact with network-connected devices.
Key idea

The methodology focuses on combining multiple layers into a single working platform, rather than evaluating each component in isolation. This enables realistic system-level behavior.

Technology stack

The PSBN Prototype integrates multiple technologies to support wireless communication, transport-layer continuity, and operational applications.

Linux-based system

The platform is built on a Linux environment, enabling control over networking behavior and integration of custom tools.

Amarisoft

Provides the wireless network emulation, including signaling, attach procedures, and real-time system logs.

MPTCP

Enables continuity at the transport layer by allowing TCP sessions to persist across multiple network paths.

TAK / WebTAK

Delivers application-layer functionality including mapping, device tracking, and communication between connected users.
