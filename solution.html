Research Methodology

This project develops a research-based Public Safety Broadband Network (PSBN) prototype to evaluate resilient mission-critical communications using multi-band cellular infrastructure and Multipath TCP (MPTCP).

Experimental Approach

Two independent cellular frequency bands are deployed to simulate a redundant PSBN environment:

  • Band 14 (LTE): Public safety spectrum.
  • Band n77 (5G NR): Mid-band high-capacity spectrum.

MPTCP Integration

Multipath TCP (MPTCP) is implemented at the transport layer to allow simultaneous data transmission over both networks. If one band fails or degrades, traffic is automatically rerouted without interrupting the active session.

Testing & Evaluation

  • Simulated band failure
  • Congestion injection
  • Latency and packet loss testing
  • High-density traffic simulation

Performance metrics include failover time, packet loss, throughput stability, latency variation, and session continuity success rate.
