Public Safety Broadband Network Prototype

A dual-band resilient PSBN architecture leveraging Band 14 LTE, Band n77 5G, and Multipath TCP (MPTCP) to ensure continuous, mission-critical connectivity under failure conditions.

Project Overview

This research project investigates the design and implementation of a resilient Public Safety Broadband Network (PSBN) prototype aligned with Canada’s national PSBN principles.

By leveraging dual-band cellular infrastructure and transport-layer multipathing, the system evaluates continuous connectivity, failover performance, and mission-critical reliability under simulated emergency scenarios.

Core Objectives

  • Implement dual-band redundancy (Band 14 & n77)
  • Enable seamless failover using MPTCP
  • Test performance under network failure conditions
  • Validate alignment with Canadian PSBN principles
