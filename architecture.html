System Architecture

The proposed PSBN prototype consists of a dual-band radio access architecture combined with an MPTCP-enabled transport layer.

Dual-Band Radio Access

Two independent cellular networks operate simultaneously:

  • LTE Band 14 (Public Safety Spectrum)
  • 5G NR Band n77 (Mid-band)

Each band connects to a core network capable of handling mission-critical traffic and secure routing.

User Equipment

  • Dual-SIM smartphones
  • UAVs transmitting real-time video
  • Mobile data terminals
  • GPS tracking devices

Traffic Flow

End devices establish concurrent subflows over both bands using MPTCP. Data is distributed across both paths to provide load balancing and seamless failover. If one path becomes unavailable, traffic continues over the remaining active link without session termination.

Resiliency Model

Redundancy is achieved through:

  • Frequency diversity
  • Transport-layer multipathing
  • Independent radio links
  • Path monitoring and dynamic rerouting
