How the system fits together

Core components

The platform is assembled from connectivity, gateway, and endpoint components. Specific hardware details can be added as your final build is locked in.

Dual-band connectivity

LTE (Band 14) and 5G (n77) provide independent network paths intended for redundancy and continuity evaluation.

Linux-based gateway

A Linux environment configured to support MPTCP and manage routing/policy behavior across interfaces.

Continuity layer (MPTCP)

MPTCP is used to explore how a session can be maintained across path changes and partial failures.

Traffic flow (high-level)

A clear narrative suitable for promotional review and technical discussion.

  1. Session starts from a client endpoint to a server endpoint using TCP.
  2. Both links are available as independent paths (LTE + 5G).
  3. MPTCP manages subflows across paths and prepares continuity behavior.
  4. A disruption occurs (loss, degradation, or transition) during an active session.
  5. Continuity is evaluated using evidence and planned metrics as testing is completed.
